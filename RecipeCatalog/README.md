# Project 2 Recipe Catalog

## Davis Larson

This is an app that allows you to store all your recipes. 

Through the use of categories, recipes can be easily divided for quick access and seperation. 

Use the search functionality to search for titles, authors, or notes among all the recipes.

Future improvements:

- Built in timer functionality
- Automatic share to messages/email
- Export to print to an AirPrinter
- Images
- Better color scheme to make it more unique

### Note

The views could be broken out much better. There is some repeated "alertView" logic that could be refactored out. 
There is also some logic for editing that is held in the view and if I had more time I would've moved that to a struct in the model as demonstrated in class. 
I am happy with how the app is currently. It works and feels professional, there are always improvements and addtions to make (of which I have ideas for) but I feel like it is already useful.
