# Project 2 Recipe Catalog

## Davis Larson

This is an app that allows you to store all your recipes. 

Through the use of categories, recipes can be easily divided for quick access and seperation. 

Use the search functionality to search for titles, authors, or notes among all the recipes.

Future improvements:

- Built in timer functionality
- Automatic share to messages/email
- Export to print to an AirPrinter
- Images
